# 📊 Data Science Internship Projects

A collection of hands-on Data Science projects focused on transforming raw datasets into meaningful insights using Python.  
These projects demonstrate skills in data analysis, visualization, customer understanding, and data preparation.

---

# 🔹 Task 1: Retail Sales Analysis

### 🎯 Objective
To explore retail sales data and identify trends, patterns, and key factors influencing sales performance.

### 🔍 Key Activities
- Performed data exploration and preprocessing
- Analyzed sales trends and customer purchasing patterns
- Created visualizations to understand business performance
- Extracted useful insights from sales data

### 📈 Result
Successfully identified sales patterns and generated data-driven insights that help in understanding customer demand and business performance.

**Technologies:** Python | Pandas | NumPy | Matplotlib | Seaborn

---

# 🔹 Task 2: Customer Segmentation

### 🎯 Objective
To analyze customer data and categorize customers into meaningful groups based on their behavior and characteristics.

### 🔍 Key Activities
- Cleaned and prepared customer data
- Studied customer attributes and relationships
- Applied clustering techniques for segmentation
- Visualized customer groups for better interpretation

### 📈 Result
Created distinct customer segments that help in understanding customer behavior and supporting targeted marketing strategies.

**Technologies:** Python | Pandas | Scikit-learn | Matplotlib | Seaborn

# 🔹 Task 3: Data Cleaning (Airbnb NYC Dataset)

### 🎯 Objective
To improve the quality of Airbnb listing data by cleaning and preparing it for further analysis.

### 🔍 Key Activities
- Identified and handled missing values
- Removed duplicate and inconsistent records
- Corrected data formats and data quality issues
- Prepared a clean dataset for analysis

### 📈 Result
Generated a structured and reliable dataset (`cleaned_AB_NYC_2019.csv`) that is ready for accurate analysis and visualization.

**Technologies:** Python | Pandas
